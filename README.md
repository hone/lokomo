# Lokomo

Heroku Buildpack Runner. Used to run a Heroku Buildpack locally.

## Installation

Add this line to your application's Gemfile:

    gem 'lokomo'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lokomo

## Usage

    $ lokomo compile /path/to/app
